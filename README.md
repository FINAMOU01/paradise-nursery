# 🌿 Paradise Nursery - Shopping Application

## 📌 Project Overview

Paradise Nursery is a front-end e-commerce web application that allows users to browse a variety of houseplants, add them to a shopping cart, and manage their selections before checkout.

This project is built using **React.js** and **Redux** for state management, and it is deployed using GitHub Pages.

---

## 🎯 Features

### 🟢 Landing Page

* Beautiful background image
* Company name and branding
* About Us section
* "Get Started" button to navigate to products

---

### 🌿 Product Listing Page

* Displays at least 6 unique houseplants
* Plants grouped into 3+ categories
* Each product includes:

  * Image
  * Name
  * Price
  * Add to Cart button
* Add to Cart button gets disabled after selection
* Shopping cart updates dynamically

---

### 🛒 Shopping Cart Page

* Displays all selected items
* Shows:

  * Total number of items
  * Total cost
* Features:

  * Increase quantity
  * Decrease quantity
  * Remove item
* Checkout button (Coming Soon)
* Continue Shopping button

---

### 🔝 Header / Navbar

* Available on Product and Cart pages
* Navigation links:

  * Home
  * Plants
  * Cart
* Shopping cart icon with dynamic item count

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/FINAMOU01/paradise-nursery.git
cd paradise-nursery
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the application

```bash
npm start
```
